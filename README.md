# Inteligencia artificial para textos científicos (TI2002S)
## Fines de Aprendizaje
Al finalizar esta Semana Tec, cada estudiante será capaz de: 
1. Elaborar escritos cuidando la coherencia, la corrección gramatical y respetando los requisitos y formatos solicitados en un escrito técnico y/o científico mediante herramientas de inteligencia artificial.
2. Utilizar de forma responsable tecnologías y medios digitales para la comunicación escrita humana y no humana mediante herramientas de inteligencia artificial.
